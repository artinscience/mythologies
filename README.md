# artinscience · mythologies

A single character's journey through a world mythology, serialized in chapters,
presented **bilingually** — the native script/tongue beside its English
translation, with period-style illustration, drawn as pen-plotter line work.
Ancient writing and ancient art are linear by nature → ideal plotter subjects.

**Culture × story × script × idiom:**
- **Greek** — the *Odyssey* / Labors of Heracles · ancient Greek · vase-art line.
- **Egyptian** — *Book of the Dead* / Osiris–Isis · hieroglyphs · tomb-register.
- **Norse / Anglo-Saxon** — *Beowulf* / Sigurd · runic & insular · knotwork.
- **Mayan** — *Popol Vuh*, the Hero Twins · Maya glyphs · codex.

Each plate: scene illustration + native-script text + English translation, like
an illuminated codex or carved stela.
